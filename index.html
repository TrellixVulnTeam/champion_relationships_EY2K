champion_relationships

Exploring the relationships between all Champions in League of Legends.

Tools Used:

Cassiopeia to interact with Riot API.

Scrapy to scrape Riot website for champion relationships not in API.

Beautiful Soup to parse the HTML from Scrapy.

R to analyze the relationships between champions.

Important Files

champions.csv - contains information about champions (name, faction, rivals, and friends) in a .csv file.

champions.json - same information in JSON.

champion_list.py - Uses Riot API to extract a formatted, ordered, list of all champions.

tutorial/spiders/champions_spider.py - Scrapy Spider that crawls Riot website for champion information not in the Riot API.
